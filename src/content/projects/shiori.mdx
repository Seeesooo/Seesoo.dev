# Shiori

Shiori is a modern-day web application for tracking light novels, webnovels, and visual novels.

> 栞 (shiori) — Japanese for bookmark.

## Why did I create it?

I got into it after tracking anime, and then I decided on tracking novels, but I found out that there aren't many good ways to track them. Novelupdates looks old, Goodreads was abandoned, and Anilist only focuses on anime with novels as an afterthought.

## Features

Shiori has many features, ranging from a progressive badge system to an activity feed and a community forum.

## Tech Stack

| Layer     | Technology                               |
| --------- | ---------------------------------------- |
| Framework | Next.js 15+ (App Router, Server Actions) |
| Language  | TypeScript                               |
| Styling   | Tailwind CSS + shadcn/ui                 |
| Database  | PostgreSQL + Prisma ORM                  |
| Auth      | NextAuth v5 (Argon2id password hashing)  |
